# Estudos de JavaScript - Nível Pleno 🚀

Este repositório armazena minha jornada de especialização em JavaScript. O foco não é apenas aprender sintaxe, mas sim **resolver problemas de negócio** com código limpo e eficiente.

- **/fundamentals**: Lógica pura, métodos de array modernos e fundamentos da linguagem.
- **/dom-manipulation**: Criação de interfaces dinâmicas e interação com o usuário.
- **/async-js**: Tratamento de APIs, Promises e fluxos assíncronos.
- **/mini-projects**: Projetos práticos simulando desafios reais do mercado.

## 🛠️ Tecnologias e Ferramentas
